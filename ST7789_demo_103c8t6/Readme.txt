	
	Exemplo completo totalmente funcional para STM32F103C8T6 com display ST7789 (240x240 IPS)
 - Relação de GPIOs utilizados:
	PA5  - SPI1_SCK		--> Clock SPI
	PA7  - SPI1_MOSI	--> Envio de dados SPI
	PB0  - ST7789_RST	--> Reset display	
	PB1  - ST7789_DC	--> Seleção de dados/comandos para o display
	PB10 - ST7789_CS	--> Chip Select
	
 - Codigo compilado (.bin) está na pasta DEBUG.
 "ST7789_demo_103c8t6.bin"
	
	
	Complete fully functional example for STM32F103C8T6 with ST7789 display (240x240 IPS)
  - List of GPIOs used:	
  	PA5  - SPI1_SCK		--> Clock SPI
	PA7  - SPI1_MOSI	--> Sending data SPI
	PB0  - ST7789_RST	--> Reset display	
	PB1  - ST7789_DC	--> Selection of data / commands for display
	PB10 - ST7789_CS	--> Chip Select
	
- Compiled code (.bin) is in the DEBUG folder.
